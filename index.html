Zero-Cost Automation

Daily Content Pipeline

A practical production loop from trend to upload. Start on free tiers, then upgrade only when the channel starts proving repeatable traction.

1

Trend Research - 6 AM Trigger

Pull the strongest AI, business, money, and tech confusion topics into the content backlog.

Make.com + ChatGPT
2

Script Generation

Generate the first draft with hook, escalation, reveal, practical lesson, and the rotating English slogan closer.

Prompt Pack + Editorial Review
3

Voiceover

Produce narration, save the final audio asset, and track the version against the video issue.

ElevenLabs or Manual VO
4

Video Assembly

Combine narration, captions, supporting visuals, pacing edits, and the Bhasad outro screen.

CapCut + Pexels
5

Thumbnail System

Use the black, red, and gold identity with a direct visual promise and a short curiosity phrase.

Canva + Image Generator
6

Description Automation

Keep descriptions in version control, dry-run them first, then update YouTube through the Data API.

GitHub Actions + YouTube API
